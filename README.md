# gamedev
Games Development

##HOW TO RUN THE GAME
- Download the project as a zip file (Press the green "code" button)
- Un zip it
- Open the folder named "Build"
- Run "gamedev.exe"
- Ignore the warnings that your PC is telling you that the file is dangerous
- You're good to go!

Controls:
- WASD to move
- F to interact 
- "Shift" to sprint

Good luck!
